○ :header
   (헤더)
   - returns : Array<Element>
   - 헤더, 실행 후 원소 배열 반환
   - 매칭되는 원소들 중 h1, h2, ... 와 같은 헤더 태그들을 모두 배열로 반환한다.
  
   
○ :animated
   (움직인다.)
   - returns : Array<Element>
   - 움직인다, 실행 후 원소 배열 반환
   - 매칭된 원소들 중 애니메이션이 동작하고 있는 것들을 모두 배열로 반환한다.
   
   
○ :contains(text)
   (포함된(내용))
   - returns : Array<Element>
   - 포함, 실행 후 원소 배열 반환
   - 매칭된 원소들 중 주어진 텍스트(text)를 포함하는 것들을 모두 배열로 반환한다.
   

○ :empty
   (비어있다.)
   - returns : Array<Element>
   - 비어있다, 실행 후 원소 배열 반환
   - 매칭된 원소들 중 텍스트가 비어있는 것들을 모두 배열로 반환한다.
   
   
○ :has(또다른선택자)
   (가진)
   - returns : Array<Element>
   - has(selector), 실행 후 원소 배열 반환
   - 매칭도니 원소들 중 주어진 것을 포함하는 것을 모두 배열로 반환한다.
   - ex) $("div:has(p)").addClass("test");
   
   
○ :parent
   (부모)
   - returns : Array<Element>
   - 부모, 실행 후 원소 배열 반환
   - 주어진 것이 부모인 요소를 모두 반아온다. 비어있는 것은 포함하지 않는다.
